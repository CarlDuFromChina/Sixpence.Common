# Sixpence.Common

> 一个基于 .net core 3.1 的通用工具框架

## 介绍

`Sixpence.Common`基于.net core 3.1，提供了丰富的工具了和扩展类，集成了开发常用的`IoC`、`Logging`和`Config`模块。使用本项目，我相信会帮助你大大提升开发效率

## 特性

+ 14个工具类
+ 8个扩展类
+ `log4net`日志集成
+ `IoC`静态类实现
+ 基于`json`的配置类封装
+ 全局上下文
